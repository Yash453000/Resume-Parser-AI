# Resume Parser AI

**Resume Parser AI** is a cutting-edge, privacy-first web application designed to instantly extract and organize data from resumes. Built with **React** and **Vite**, it processes **PDF** and **DOCX** files entirely on the client-side, ensuring that sensitive personal data never leaves the user's device.

## Key Features

*   **Instant Parsing**: Automatically extracts names, contact details, skills, and professional classifications using advanced regex and keyword matching.
*   **Privacy-Centric**: Zero server-side processing means complete data security. Your resume never leaves your browser.
*   **Futuristic UI**: Features a stunning **Cyberpunk/Futuristic** aesthetic with neon accents, glassmorphism, and a custom animated background. Includes both **Dark** and **Light** futuristic themes.
*   **User-Friendly**: Simple drag-and-drop interface with immediate results display.

## Tech Stack

*   **React + Vite**: For a blazing fast frontend experience.
*   **PDF.js**: For robust PDF text extraction.
*   **Mammoth.js**: For parsing DOCX documents.
*   **Lucide React**: For modern iconography.

This tool showcases the power of modern frontend technologies to deliver AI-like utility without the need for complex backend infrastructure. Perfect for recruiters and developers looking for a fast, secure parsing solution.
